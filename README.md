# 🧺 Laundry App

A modern Flutter application designed to streamline laundry services, from booking pickups to tracking order status.

## ✨ Features

*   **User Authentication**: Secure login and sign-up (Firebase/Supabase).
*   **Service Selection**: Choose between wash, dry-clean, or iron services.
*   **Order Tracking**: Real-time updates on your laundry status.
*   **Schedule Pickups**: Choose convenient time slots for pickup and delivery.
*   **Secure Payments**: Integrated payment gateway for seamless transactions.

## 🚀 Getting Started

### Prerequisites
* Flutter SDK (Latest Version)
* Dart SDK
* An Android/iOS Emulator or Physical Device

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/laundry.git
   ```
2. Navigate to the project directory:
   ```bash
   cd laundry
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
